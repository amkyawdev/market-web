# 🛒 AmkyawDev Market

A modern e-commerce platform for selling websites and web applications with built-in admin authentication system.

![AmkyawDev Market](https://img.shields.io/badge/AmkyawDev-Market-blue?style=for-the-badge&logo=github)
![Version](https://img.shields.io/badge/version-1.0.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)

---

## 📋 Features

### 🔐 Admin Authentication
- Secure login system with email/password
- Protected admin panel routes
- Session management with localStorage
- Glassmorphism UI design

### 🛍️ Market Page
- Modern product showcase for websites
- Category filtering (Websites / Web Applications)
- Demo links to GitHub repositories
- One-click order system

### ⚙️ Admin Panel
- **Orders Management** - View, complete, and delete orders
- **Visitor Tracking** - Monitor user activity and login attempts
- **Website Management** - Add and manage products

### 🎨 UI/UX
- Three.js animated background
- Glassmorphism design
- Responsive layout
- Smooth animations

## �Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Git

### Installation

```bash
# Clone the repository
git clone https://github.com/amkyawdev/market-web.git

# Navigate to project directory
cd market-web

# Open in browser
# Simply open index.html in your browser
# Or use a local server:
python -m http.server 8080
```

## 📂 Project Structure

```
market-web/
├── index.html              # Main entry point
├── assets/
│   ├── style.css          # Global styles
│   └── favicon.svg        # Site favicon
├── data/
│   └── website.json       # Website data
└── src/
    ├── core/
    │   ├── app.js         # App entry point
    │   ├── auth.js        # Authentication logic
    │   ├── engine.js      # Three.js engine
    │   └── router.js      # Client-side router
```
